# Bank Marketing Campaign Analysis Dashboard

## Project Overview
This project presents an interactive Power BI dashboard built using the Bank Marketing Campaign Dataset from the UCI Machine Learning Repository. The dashboard analyzes marketing funnel performance, customer conversion behavior, and campaign effectiveness.

The main objective of this project is to identify factors that influence customer subscriptions and provide data-driven recommendations to improve marketing campaign performance.

---

## Dataset Information
- Dataset: Bank Marketing Campaign Dataset
- Source: UCI Machine Learning Repository
- Total Records: 45,211 customers
- Tool Used: Microsoft Power BI

Dataset Link:
https://archive.ics.uci.edu/dataset/222/bank+marketing

---

## Business Problem
Banks invest heavily in marketing campaigns, but not all customers convert successfully. This project analyzes customer behavior, campaign outcomes, and funnel performance to identify:
- High-performing marketing channels
- Customer conversion trends
- Drop-off points in the marketing funnel
- Customer segments with high conversion potential

## Data Cleaning & Transformation
The dataset was cleaned and transformed using Power Query in Power BI:
- Removed null and unknown values
- Corrected data types
- Renamed columns
- Created Age Group categories
- Created conversion metrics
- Standardized text values
- Added Month Number sorting

---

## Key Insights
- Cellular contact methods achieved the highest conversion rates.
- Customers aged 25–44 showed stronger conversion performance.
- Longer call durations increased subscription success.
- Previous successful campaign outcomes improved future conversion likelihood.
- Most customers dropped off after initial contact.

---

## Recommendations
- Focus marketing efforts on cellular communication channels.
- Improve first-contact engagement strategies.
- Target high-converting customer segments.
- Personalize campaigns using previous customer behavior.
- Reduce repeated contact attempts for low-engagement customers.
